<br>
We have found setting the GPU affinity is an absolute must for multi GPU systems on windows.<br>
<br>
Basically from what I understand, windows will by default dispatch the OpenGL commands to what it thinks is the fastest GPU in the system, then copies the result to the other GPU(s). Linux by default dispatches the commands to the GPU attached to the display the window is on......so setting the GPU affinity is most likely only worth while if you are using windows.<br>
