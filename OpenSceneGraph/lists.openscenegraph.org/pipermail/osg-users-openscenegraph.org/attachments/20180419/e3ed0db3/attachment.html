 (i.e. getOrCreateUniform(“some_string” + std::to_string(4), … )<o:p></o:p></li><li class="MsoListParagraph" style="margin-left:0in;mso-list:l0 level1 lfo1">You can NOT index an array when setting a uniform’s value (i.e. uniform->set(num_array[0])).. It needs to be copied first (i.e. int x = num_array[0]; uniform->set(x);<o:p></o:p></li></ol>
