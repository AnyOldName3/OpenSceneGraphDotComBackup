OSG (windows at least) supports off screen rendering without any window flashing on the screen by using pbuffer.<br>
<br>
When you create the Traits object, set traits->pbuffer = true;<br>
<br>
See example osgscreencapture, see pbuffer-only parameter.<br>
<br>
Start the example with osgscreencapture --pbuffer-only 512 512 cow.osg<br>
